# SudokuSolver (WEBSITE IN PROGRESS)

<h2>Description</h2>
The aim of this project is to write a program that can solve any (solve-able) sudoku puzzle. Three sudoku.txt example files, with solutions for two of them (designed by Professor David Byrd of Bowdoin College) are contained within the .zip file. The "runner" class is titled "SudokuTest", and javadoc comments exaplin the function
<br />


<h2>Language</h2>

- <b>Java (Oracle OpenJDK Version 21)</b>

<h2>Environments Used </h2>

- <b>IntelliJ IDEA 2023.3.6 </b> (21H2)
